llm serving 최적화 패턴을 적용하기 위한 나만의 simple_vllm 

## Install it from PyPI

```bash
pip install simple_vllm
```

## Usage

```py
from simple_vllm import BaseClass
from simple_vllm import base_function

BaseClass().base_method()
base_function()
```

```bash
$ python -m simple_vllm
#or
$ simple_vllm
```
